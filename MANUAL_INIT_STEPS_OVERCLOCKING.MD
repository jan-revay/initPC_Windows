# Manual steps when overclocking resp. binning CP parts

- install all drivers and SW first
- run stress tests with default BIOS settings
- run AMD curve optimizer (Ryzen master)
- find max memory freq
- ...

## Programs to install

- Zen timings
- prime95
- y-cruncher
- ryzen master
- HWinfo64  (different from HWmonitor)
- OCCT
